# 🚗 RentWheels

**RentWheels** is a modern car rental web application where users can easily browse, book, and manage car rentals online.  
It connects users with local car owners or rental providers in a simple, fast, and user-friendly way.

---

## 🌐 Live Website
https://rentwheels-f11f0.web.app

---

## ✨ Main Features

- 🔐 **Secure Authentication**  
  Login and register using email/password or Google (Firebase Authentication).

- 🚘 **Browse Cars Easily**  
  Explore a wide range of cars with images, pricing, and detailed information.

- 📅 **Instant Booking System**  
  Select your preferred car, choose dates, and confirm booking within seconds.

- 🧑‍💼 **Private Dashboard**  
  Logged-in users can manage their bookings and listings securely.

- 🎨 **Modern UI/UX**  
  Smooth animations using Framer Motion, Lottie, and Swiper for better experience.

- 🔔 **Real-Time Feedback**  
  Instant success/error messages using Toastify and SweetAlert2.

---

## 🛠 Tech Stack

**Frontend:**
- React 19

- React Router DOM
- Framer Motion

- React Icons
- React Toastify / Sonner
- Swiper / Slick Carousel
